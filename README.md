# 임베디드 시스템 수업 프로젝트 

지능형 임베디드 시스템 수업 시간에 진행한 실습 및 과제 코드 저장소입니다.

## 🛠 개발 환경
* **IDE**: Keil Studio (Cloud/Desktop)
* **Target**: ARM Cortex-M 기반 보드 (NUCLEO-F401RE 등)
* **Language**: C/C++ (Mbed OS)

## 📁 주요 내용
* 수업 시간별 실습 코드 (GPIO, UART, Interrupt 등)
* 기말/중간 프로젝트 소스 파일

## 🚀 실행 방법
1. Keil Studio 접속 및 로그인
2. `Import Project` 클릭 후 이 저장소 주소 붙여넣기
3. 타겟 보드 연결 후 `Build & Run`
